The moderation status can be get via the remixes of project data. If your project is not remixed, our auto bot will remix this with the account "RemixMaster", perform the search and then delete the remix. Also, with this tool you can see the data of any project, not only yours.

Data that you can get of ANY project:

  • Moderation status
  • If the project is visible on searches (NFE)
  • f the project is shared

This also works with unshared projects, in this case, you will also get the project title, author and the parent project (what is the remixed project). This data will be soon added to the unshared viewer, if the bot can remix unshared projects we (think that yes).

We are working in the remix bot, so if the anwser is "no data", we will remix it. Also, in this API there are available 4 dates related to the project. We are investifgating what they could be. There are "ctime", "datetime_created", "mtime" and "datetime_shared" and the value (I think that) are timestamps, but noone of that dates coincides with th project data shown in the project page. I'm going to test it with another project, maybe this is a little error. I think that ctime = creation time - datetime_created = same that ctime - mtime = modification time - datetimeshared = share time.

Also, this API has errors: it shows old project data. With the projects that I tested, it shows a wrong title (an old title that I changed) and a wrong count of favourite and loves, so maybe the dates are also bad.

Also, this api shows unshared remixes, so it will work with the remix bot. I won't share the api link yet
